10.

Stwórz obiekt o nazwie osoba z polami: imie, nazwisko, wiek oraz czyUczen.
Uzupełnij obiekt przykładowymi wartościami, np.: Jan, Kowalski, 16, true
Do obiektu dopisz metodę o nazwie wypisz, która powinna działać następująco:
- jeśli pole czyUczen jest ustawione na true - wypisz w tagu "zad10" słowa: "Uczeń: - <tu_imię> <tu_nazwisko>"
- jeśli pole czyUczen jest ustawione na false - wypisz w tagu "zad10" imię, nazwisko i wiek w postaci: <imię> <nazwisko> lat: <wiek>
