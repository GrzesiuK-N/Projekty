10.

Stwórz obiekt o nazwie zwierze z polami: gromada, gatunek, czyZagrozone oraz populacja.
Uzupełnij obiekt przykładowymi wartościami, np.: ptaki, orzeł bielik, false, 40000.
Do obiektu dopisz metodę o nazwie szczegoly, która powinna działać następująco:
- jeśli pole czyZagrozone jest ustawione na true - wypisz w tagu "zad10" słowa: "zagrożone zwierzę - <tu_nazwa_gatunku>"
- jeśli pole czyZagrozone jest ustawione na false - wypisz w tagu "zad10" gromadę, gatunek i populację, np. "ptaki, orzeł bielik, 40000"

po kliknięciu przycisku "uruchom" wywolaj metodę szczegoly dla oiektu zwierze.
